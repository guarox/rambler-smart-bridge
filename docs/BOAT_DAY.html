Dashboard States

○ NO DATA

No Instruments

Pi connected, PICAN-M not yet on N2K backbone. Values show —. This is the correct state when first connecting.

◌ SK…

Signal K Connecting

URL set, WebSocket connecting or reconnecting. Normal during boot. Becomes LIVE when instruments flow.

● LIVE

Instruments Live

N2K data flowing through PICAN-M → Signal K → dashboard. All values are real.

Starlink Setup on Boat

✅
XTAR-Link EL6 V3 has a built-in ethernet port. No separate Starlink Ethernet Adapter needed. Just run an ethernet cable from XTAR-Link to Pi eth0.

Connection

1

Run ethernet from XTAR-Link ethernet port → Pi eth0

2

Pi picks up Starlink IP via DHCP. No config needed.

3

Crew on Rambler_Net gets internet via NAT automatically.

Without Starlink (Race Mode)

Everything still works with no internet. Pi broadcasts Rambler_Net, dashboard loads from Pi, Signal K reads instruments. Crew just has no internet — fine for racing.

XTAR-Link also powers Starlink

The EL6 V3 takes 12V from your boat battery and converts it to power the Starlink dish — eliminating the need for 110V shore power. Single cable to the Starlink dish handles both power and data.
